# LifeBuddy POC\n\nThis repository contains a simple proof of concept for the Life Operating System described in `PRD.MD`.\n\n## Structure\n- `frontend/`: React + Vite dashboard for visualizing ingested thoughts and calendar events\n- `backend/`: Node server that ingests local JSON files and exposes a summary API\n\n## Dummy data sources\n- `thoughts.json`\n- `calendar.json`\n\n## Run the POC\n### Backend\n```bash\ncd backend\nnode server.js\n```\n\n### Frontend\n```bash\ncd frontend\nnpm install\nnpm run dev\n```\n\nThe frontend expects the backend on `http://localhost:4000`.\n
